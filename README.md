- JavaScript (vanilla, no frameworks)
- GitHub Pages hosting
- No build step required

- `pages/` — Subpages (about, contact, industries, join, team, privacy, terms, 404)
- `assets/imp-logo.png` — Brand logo
- `assets/favicon.svg` — Favicon
